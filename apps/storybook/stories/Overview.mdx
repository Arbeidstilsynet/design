# Arbeidstilsynet Design

The Norwegian Labour Inspection Authority's design system, based on [designsystemet.no](https://www.designsystemet.no).

## Packages

- [@arbeidstilsynet/design-react](https://www.npmjs.com/package/@arbeidstilsynet/design-react) - React implementation of components
- [@arbeidstilsynet/design-css](https://www.npmjs.com/package/@arbeidstilsynet/design-css) - Styling for components
- [@arbeidstilsynet/design-theme](https://www.npmjs.com/package/@arbeidstilsynet/design-theme) - Arbeidstilsynets styling theme(s)
